About the API

Welcome to the Compendium Data Viewer, a tool that will help you visualize each and every entry of the bestiary from the game The Legend of Zelda: Breath of the Wild - directly on your computer or smartphone screen !


This website uses Aarav Borthakur's Hyrule Compendium API as its main source of data. It stores the data of the 394 entries of the compendium, which are all the monsters, equipment, treasures, creatures and materials that you can find on the vast continent of Hyrule.

How to use the API

As the API doesn't require any key or token to send responses, a simple HTTP GET request suffices to receive data.

For exemple, here's the request that has generated the card above!
GET https://botw-compendium.herokuapp.com/api/v2/entry/8

Endpoints

    

        
        

            
 Single Entry

            

                
Retrieving a unique item can be done using the request mentionned above.

                Please note that you can specify the entry's name instead of its ID in the request
                

                    GET https://botw-compendium.herokuapp.com/api/v2/entry/moblin
                

            

        

        
        
            
 All entries from a category

            

                In addition to the entry endpoint mentionned above, you can retrieve data from the API by their category ("monsters", "equipment", "materials", "creatures" and "treasures")
                

                    GET https://botw-compendium.herokuapp.com/api/v2/category/monsters
                

            

        

        

            
 All entries

            

                It is also possible to retrieve all 389 items using the following request:
                

                    GET https://botw-compendium.herokuapp.com/api/v2/all
                

            

        

        

            
 Mastermode

            
If you want to retrieve entries specific to the mastermode (golden enemies, for example), this must be done by slightly altering the URL Request.

            

                To retrieve one entry exclusive to the mastermode:
                

                    GET https://botw-compendium.herokuapp.com/api/v2/master_mode/entry/golden_lynel
                

            

            

                To retrieve all items from the mastermode:
                

                    GET https://botw-compendium.herokuapp.com/api/v2/master_mode/all
                

                
Please note that this endpoint will only retrieve all the entries relative to the mastermode and not the full mastermode compendium.
